paisa
=====

<small><i>paisa translates to money in Nepali</i></small>

a small expense tracker

Features
--------

- Local only
- Export to CSV or XLSX
- Periodic export support
- Easy to insert data (with `@`, `#`, and `$` everything can be specified)
